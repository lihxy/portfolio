Shape Generator

This is an interactive animation. It generates a shape based on how many sides the user wants. In addition, random shapes can be generated by clicking a certain object.

Paddle Ball

This is a game created in Scratch. The objective is to move the paddle to bounce the ball off of the blocks to gain points. If the ball fails to touch the paddle after it hits a block, you lose.

Get to the Igloo

A scrolling game created on Scratch. The objective is to avoid the obstacles to get the Penguin home without it freezing to death. Everytime you hit an obstacle, the temperature decreases and if it reaches -30 you lose.
